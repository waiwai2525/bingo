# ビンゴ司会システム

社会情報実践 制作物 No. 1

## 要件定義

ビンゴで番号をランダムに提示・記録するシステム

### ビンゴの基本ルール

- 1から75までの範囲の数字をランダムに選び提示する．
- 同じ数字は選べない．
- 提示済みの番号は順番に記録して表示する．

### 技術仕様

- 単独で動作するアプリ（Webアプリも可）とする．
- ゲーム開発環境（Unityなど）は利用不可とする．

#### 動作プラットフォーム

macOS, Windows, Linux（Electron.jsのサポートに準ずる）

#### 開発プラットフォーム

macOS, WebStorm

#### 開発言語

Typescript

#### 利用ライブラリ

Electron, Vite, React, Semantic UI
